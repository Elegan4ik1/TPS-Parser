🚀 Профессиональный полуавтоматический парсер truepeoplesearch.com с графическим интерфейсом
Хотите получать максимум данных о людях быстро, без рутины, но при этом контролировать процесс?
Этот скрипт создан для тех, кто ценит своё время и качественный результат, но понимает, что полная автоматизация в условиях жёстких антибот-систем невозможна без偶尔 ручного вмешательства.

🔥 Что умеет скрипт (автоматически)
✅ Автоматический поиск профилей по базе truepeoplesearch.com (США)
✅ Обход Cloudflare Turnstile – капча решается через платный сервис CapMonster Cloud (поддержка Turnstile, готовимся к reCAPTCHA)
✅ Умная ротация IP – встроенная поддержка Mullvad VPN + список SOCKS5/HTTP прокси (выбор за вами)
✅ Синхронизация обработанных номеров между несколькими копиями скрипта – все успешные и пустые номера сохраняются в облачной базе данных (входит в стоимость лицензии). Это позволяет нескольким пользователям работать с общей базой и исключать повторные проверки.
✅ Сбор всех подходящих людей – скрипт находит всех, кто подходит по возрасту (настраивается в конфиге) и парсит каждого в отдельной вкладке
✅ Подробные данные – имя, возраст, дата рождения, округ (County), текущий и предыдущие адреса
✅ Фильтрация дубликатов – номера из локальных файлов и облачной БД не обрабатываются повторно
✅ Сохранение очереди – при остановке скрипт сохраняет необработанные номера и при следующем запуске восстанавливает их
✅ Telegram-уведомления – о найденных профилях, пустых номерах и завершении работы (опционально)

🖥️ Графический интерфейс (GUI)
Две адаптивные колонки: слева – настройки, справа – лог выполнения в реальном времени

Все параметры редактируются в окне (сохраняются в config.json)

Кнопки «Запустить» / «Остановить», индикатор текущего этапа, статусная строка

Лог выводит каждое действие – ввод номера, решение капчи, найденные профили, бан, смену IP

Автоматическая очистка и прокрутка лога – ничего не пропустите!

⚠️ Важно: полуавтоматический режим и дополнительные расходы
Сайт truepeoplesearch.com активно борется с ботами. Наш скрипт автоматизирует 95% работы, но в некоторых ситуациях требуется ваше участие:

🔹 Сбои VPN/прокси – если Mullvad или прокси временно «отваливаются», может потребоваться переключить страну или перезапустить VPN вручную.
🔹 Сложные капчи – CapMonster решает Turnstile отлично, но иногда Cloudflare выдаёт особо хитрые задания, которые лучше пройти вручную один раз, чтобы разблокировать сессию.
🔹 «Зависания» браузера – при нестабильном интернете или ошибках Playwright окно браузера может зависнуть; достаточно закрыть его и нажать «Запустить» снова – скрипт продолжит с того же места.

Это не баги, а особенности работы с современными антибот-системами. Мы даём вам прозрачный интерфейс и подробный лог, чтобы вы всегда видели, что происходит, и могли вмешаться, когда это необходимо.

💸 Необходимые платные сервисы
Все сервисы, кроме VPN, уже входят в стоимость лицензии. Вам нужно будет оплатить только:

CapMonster Cloud (или любой другой сервис с API для решения Turnstile).
Стоимость: примерно $0.5–2 за 1000 решённых капч, в зависимости от тарифа.
Баланс пополняется отдельно, ключ API вводится в настройках.

Mullvad VPN (опционально, если вы не используете прокси).
Стоимость: €5 в месяц. Можно заменить на любой другой VPN или прокси.

Все остальные компоненты (Playwright, aiohttp, rich, база данных и т.д.) устанавливаются автоматически и бесплатны.

📋 Краткая инструкция по установке
Убедитесь, что установлен Python 3.10 или выше (скачать с python.org, при установке отметьте «Add Python to PATH»).

Скачайте архив со скриптом и распакуйте в удобную папку (например, C:\parser).

Запустите файл install.bat от имени администратора – он создаст виртуальное окружение, установит все зависимости и браузеры.

После установки запустите программу двойным щелчком по gui.pyw или через ярлык (будет создан автоматически).

В настройках укажите:

Ключ CapMonster (API ключ из личного кабинета)

При необходимости – данные для Mullvad VPN или файл с прокси

Ваш Telegram-бот токен и Chat ID (если нужны уведомления)

Поместите номера в файл phones.txt (по одному номеру в строке).

Нажмите «Запустить» и наблюдайте за процессом в логе.
